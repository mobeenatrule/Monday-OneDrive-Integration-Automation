# Monday.com → OneDrive Integration Automation

This project automates file and folder management between Monday.com and OneDrive using Make.com.

It ensures that files related to tasks or items in Monday.com are automatically organized, stored, and managed in OneDrive without manual effort.

---

## 🚀 Overview

This automation connects Monday.com with OneDrive to:

- Create folders dynamically  
- Upload and manage files  
- Organize documents based on workflow data  
- Maintain structured storage system  

---

## ⚙️ How It Works

1. A webhook triggers the workflow  
2. Data is fetched from Monday.com  
3. Folder structure is prepared dynamically  
4. Iterator processes multiple items/files  
5. OneDrive creates folders  
6. Files are retrieved and processed  
7. Files are uploaded to the correct location
